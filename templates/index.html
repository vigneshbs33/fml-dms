Neural Network Flow

f(x) = (f₃ ∘ f₂ ∘ f₁)(x)

Composition: Each layer is a function. The neural network chains them: Output = f₃(f₂(f₁(Input))). This is function composition.
Input x 64 features

f₂ · ReLU · 16 units

f₃ · Softmax · 10 classes
Output ŷ 10 classes

🕊️ Pigeonhole Principle

We are mapping 2⁶⁴ possible inputs → just 10 output classes. Since the domain is astronomically larger than the codomain, many distinct drawings must produce the same prediction. Collisions are mathematically guaranteed.
